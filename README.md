## Weather Hub
## Description

A weather dashboard featuring dynamically updated HTML and CSS. Search for current weather and a five day forecast in any city. Saved searches are persistent. 
## Built With

* JavaScript, jQuery
* HTML, CSS, Bootstrap
* Day.js
* OpenWeather One Call API
